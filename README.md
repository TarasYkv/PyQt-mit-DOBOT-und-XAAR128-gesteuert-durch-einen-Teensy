# PyQt-mit-DOBOT-und-XAAR128-gesteuert-durch-einen-Teensy
XAAR 128 mit DOBOT Magician und PYTHON. Die Nutzeroberfläche (GUI) ist mit Python (PyQt) realisiert. Von der GUI werden Befehle an einen Microcontroller (Teensyduino) verschickt. Ebenso wird ein ausgewähltes Bild auf einer Speicherkarte abgelegt.
